# 歡迎光臨「迷路的分店」：一場考驗運氣與常識的迷宮冒險
>「這不僅僅是走出迷宮，而是一場關於迷失與回家的旅程。」

這是一款基於 **Python (Pygame)** 開發的惡搞風迷宮生存遊戲。結合了 **戰爭迷霧**、**冷知識問答** 以及 **道德綁架（？）** 的要素，考驗你的記憶力、運氣以及對中央大學的熟悉程度。

## 遊戲特色

這不是普通的迷宮，這裡每一步都充滿了驚喜（與驚嚇）：

* **動態生成的迷宮**：每次重來都是全新的地圖，背板是沒有用的。
* **視野限制系統**：從第一關的全知視角，到後面伸手不見五指的迷霧模式，體驗路癡的絕望。
* **NCU 冷知識問答**：遇到怪物不用拔劍，請準備好你的大腦。答錯？請重修（回起點）。
* **道德考驗**：你真的忍心把那隻跟在屁股後面的小狗丟在迷宮裡嗎？

## 遊戲機制與怪物圖鑑

### 關卡挑戰
| 關卡 | 名稱 | 難度描述 |
|:---:|:---|:---|
| **LV 1** | **新手村** | 享受這最後的全知視野吧，這裡只有簡單的問答怪和傳送坑。 |
| **LV 2** | **迷霧驚魂** | **關燈了！** 你只能看到周圍 5x5 的範圍，但凡走過必留下痕跡（足跡保留）。 |
| **LV 3** | **深淵漫步** | **路癡地獄**。視野5x5但**不保留路徑**。還可能要護送一隻玻璃心的小狗回家。 |

### 生存圖鑑
| 圖示顏色 | 名稱 | 能力與應對方式 |
|:---:|:---|:---|
| 🟥 **紅色** | **中央常識題庫**<br>(問答怪) | 隨機移動的紅色方塊。撞到後觸發「中央大學隨堂考」。<br>**答對**：怪物消失，繼續前進。<br> **答錯**：**被當掉！直接傳送回起點重修。** |
| 🟪 **紫色** | **免費機票**<br>(隨機傳送陷阱) | 地上的隱形坑。踩到後會被**隨機傳送**到地圖任意角落。<br>可能送你去終點，也可能送你去填海。 |
| 💗 **粉色** | **半夜的情侶**<br>(閃光怪) | 佔據兩格的巨大障礙物。雖然很刺眼又擋路，但**主動撞上去**可以利用他們的閃光**照亮全圖 4 秒**！<br>*(單身狗的勝利)* |
| 🦴 **膚色** | **心碎小狗**<br>(護送任務) | **Level 3 限定**。撞到牠後會觸發跟隨模式。<br>**通關條件**：必須帶著牠**走回起點 (1,1)** 完成「送回家」任務，才能去終點過關。<br>如果不送牠回家就想跑？系統會判定你沒良心而不讓你過關。 |

## 操作指南 (Controls)

程式碼裡藏了一些好用的功能，別說我沒告訴你：

| 按鍵 / 操作 | 功能 | 備註 |
|:---|:---|:---|
| **方向鍵 (↑↓←→)** | 移動角色 | 基礎中的基礎。 |
| **空白鍵 (Space)** | 確認對話 / 關閉訊息 | 看到小狗說話請按空白鍵讓牠閉嘴。 |
| **滑鼠點擊** | 選取輸入框 | 回答問題時，請先點擊白色輸入框。 |
| **鍵盤打字 + Enter** | 回答問題 | 支援數字及英文輸入，打完答案記得按 **Enter** 送出。 |

## 🛠️ 開發環境與執行 (Setup & Run)

本專案指定使用 **Visual Studio Code** 搭配 **Python 3.11** 進行開發與執行。

### 1. 環境準備 (Prerequisites)
請確保你的電腦已安裝以下軟體：
* **[Python 3.11](https://www.python.org/downloads/)**：請下載並安裝 3.11 版本（安裝時記得勾選 *Add Python to PATH*）。
* **[Visual Studio Code (VS Code)](https://code.visualstudio.com/)**：推薦的程式編輯器。
* **VS Code Python 擴充套件**：打開 VS Code 左側擴充欄 (Extensions)，搜尋並安裝 Microsoft 出品的 `Python` 套件。

### 2. 安裝依賴 (Install Dependencies)
1. 用 VS Code 開啟本專案資料夾。
2. 開啟下方終端機 (**Terminal** > **New Terminal**)。
3. 確認 Python 版本（選用）：
   ```bash
   python --version
   # 確保顯示為 Python 3.11.x
4. 輸入以下指令安裝 pygame 套件：
   ```bash
   pip install pygame
